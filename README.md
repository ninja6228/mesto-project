## Описание
Интерактивный одностраничный сайт на нативном JS с адаптацией под различные виды устройств.  
Семантическая, кроссбраузерная, валидная вёрстка. 


## Стек технологий
 * JavaScript 
 * ООП (Классы)
 * БЭМ
 * Media queries
 * Flexbox
 * Grid Layout
 * Webpack
 * Npm
 * API
 * Парное программирование


## Рефакторинг кода
Функционал полностью переведен на ООП
